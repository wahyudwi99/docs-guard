- [-] BUG-001 (Terkait TASK-003): Fungsi `pdfDoc.encrypt` di `hooks/useFileExport.ts` tidak ada dalam pustaka `pdf-lib` (versi 1.17.1). Aplikasi akan mengalami error "pdfDoc.encrypt is not a function" saat mencoba mengekspor PDF dengan proteksi password. Cara reproduksi: Unggah PDF, masukkan password (sebagai user Pro), dan klik Download.
- [-] BUG-002 (Terkait TASK-004): Pembersihan metadata untuk gambar tidak selektif. Hasil ekspor selalu berupa PNG yang menghapus semua metadata secara otomatis melalui canvas, sehingga opsi selektif tidak berfungsi. Selain itu, ekspor JPG yang diminta dalam spesifikasi tidak tersedia. Cara reproduksi: Unggah JPG, pilih opsi metadata tertentu, ekspor sebagai gambar, dan periksa hasilnya yang selalu PNG tanpa metadata.
- [-] BUG-003 (Terkait TASK-002): Tombol "Blur Mode" tidak memiliki indikasi visual fitur premium (seperti ikon kunci) meskipun fitur ini memiliki batasan freemium. Hal ini tidak konsisten dengan fitur premium lainnya (Password & Metadata) yang memiliki label "Pro Only". Cara reproduksi: Buka tab Design dalam mode free, tombol Blur Mode terlihat seperti fitur gratis sepenuhnya.
- [-] BUG-004 (Terkait TASK-006): Listener `addCustomerInfoUpdateListener` di `hooks/useSubscription.ts` tidak dibersihkan saat unmount, yang berpotensi menyebabkan kebocoran memori. Cara reproduksi: Review kode pada `hooks/useSubscription.ts` di bagian `useEffect` untuk listener.
- [-] BUG-005 (Terkait TASK-008): Penggunaan `window.location.origin` sebagai basis `callbackUrl` di Capacitor iOS berpotensi menyebabkan error redirect pada Google OAuth. Cara reproduksi: Jalankan aplikasi di simulator iOS, coba lakukan alur langganan yang memicu redirect login.
- [-] BUG-006 (Terkait TASK-008): Mekanisme `autoSubscribe` di `app/page.tsx` tidak memiliki penanganan jika data `packages` dari RevenueCat terlambat dimuat, menyebabkan proses otomatis terhenti. Cara reproduksi: Simulasikan koneksi lambat, lakukan login via paywall, dan lihat apakah proses purchase terpanggil otomatis setelah kembali ke halaman utama.
- [-] BUG-007 (Terkait TASK-008): Variabel `authStatus` di `app/page.tsx` didefinisikan tetapi tidak digunakan (unused variable). Cara reproduksi: Jalankan `npm run lint` atau periksa baris 42 pada `app/page.tsx`.
