# TASK-005: Pengembangan Layar Kebijakan Privasi "Amnesia Total"

## Deskripsi
Membuat halaman Kebijakan Privasi yang profesional untuk meyakinkan pengguna bahwa data mereka aman.

## Spesifikasi
1. **Poin Utama:**
   - **Amnesia Total:** Tidak ada riwayat dokumen yang disimpan.
   - **100% On-Device:** Semua pemrosesan terjadi secara lokal di perangkat pengguna.
   - **Zero Cloud:** Tidak ada penyimpanan cloud atau upload file ke server kami.
   - **Zero Selling:** Tidak ada penjualan data pengguna ke pihak ketiga.
2. **Desain:** Gunakan UI yang bersih, profesional, dan sangat mudah dibaca.
